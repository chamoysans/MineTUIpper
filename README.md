
# `MineTUIIper`
A TUI (Terminal User Interface) for Minesweeper.

## Installation

1. Install Python if you havenâ€™t already.
2. Download `minesweeper.py` and `requirements.txt`.
3. Open a terminal and navigate (`cd`) to the folder containing `requirements.txt`.
4. Run the following command to install dependencies:

   ```bash
   python -m pip install -r requirements.txt
   ```

## How to run the game
1. Open a terminal and navigate (`cd`) to the folder containing `minesweeper.py`.
2. Run the game with: `python minesweeper.py`
